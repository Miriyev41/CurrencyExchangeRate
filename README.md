## Project Description
A network-based system simulating an online currency exchange office. The system utilizes a WCF Web Service backend to retrieve real-time currency exchange rates from the National Bank of Poland (NBP) API. It features a progressive architecture including business logic for currency exchange, database persistence for user accounts and transactions, and a client application for user interaction.
